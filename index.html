              خزانة الأدوات السحرية

              ✦

              لا يقتصر التصميم في نظري على رصف العناصر عبر الشاشات؛ بل هو رحلة سبرٍ لأغوار النفس البشرية أولاً.

              Figma

              العصا الأساسية لرسم العوالم ونسج الواجهات.

              Miro

              ساحة التأمل البيضاء لرسم مسارات وتجارب المستخدمين.

              Maze

              متاهة الاختبار لاستشراف تفاعل المستخدمين وجمع أصداء تجاربهم.

              Hotjar

              عين البصيرة التي ترصد الخرائط الحرارية وتكشف مسارات التفاعل.

              Cursor

              الجسر الأثيري الذي يربط بين عوالم التصميم وشيفرات الكود.

              Gemini

              بلورتي السحرية لاستحضار المعرفة واستلهام الأفكار.

              Stitch

              نول النسيج الذي يحول التصاميم إلى مكونات محكمة.

              Mobbin

              مخطوطتي المفتوحة لاستكشاف أفضل الممارسات المعاصرة.

              ChatGPT

              قريني التحليلي لفك طلاسم أبحاث المستخدمين.

              Lovable

              عصاتي السحرية لتحويل التصاميم إلى أكواد واقعية بلمحة بصر.

              VS Code

              محرر الأكواد الأنيق لنسج التعاويذ البرمجية وتطوير الواجهات.

              Notion

              مذكرتي السحرية لتنظيم الأفكار وتوثيق رحلة التصميم.

              GitHub

              مستودع الأسرار لحفظ الأكواد ومشاركة التعاويذ مع العالم.

              Framer

              منصة التشكيل السريع لبناء واجهات تفاعلية تخطف الأنفاس.
